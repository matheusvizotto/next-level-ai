# Prompt 03, Brief de Conteúdo

**Quando usar:** Antes de criar qualquer peça de conteúdo, seja post, vídeo, artigo ou email.

---

## Prompt

```
Preciso criar um brief de conteúdo.

Contexto:
- Canal: {{Instagram Reels / LinkedIn / YouTube / Newsletter / Blog}}
- Tema ou ideia inicial: {{O QUE VOCÊ QUER FALAR}}
- Público-alvo: {{QUEM VAI VER ISSO}}
- Objetivo: {{informar / gerar leads / vender / construir autoridade / engajar}}
- Tom: {{formal / casual / técnico / inspiracional}}

Com base nisso, crie um brief com:
1. Ângulo principal (a ideia central, o "por que isso vai funcionar")
2. Hook (a primeira frase ou cena, o que vai fazer a pessoa parar de rolar)
3. Estrutura do conteúdo (pontos principais em ordem lógica)
4. CTA (o que eu quero que a pessoa faça depois de consumir)
5. Palavras-chave ou frases que preciso incluir para distribuição orgânica
6. Uma variação alternativa de ângulo caso o primeiro não funcione

Salve o brief em 04 Resources/textos/{{CANAL}}/YYYY-MM-DD-{{slug-do-tema}}.md
```

---

## O que esperar

Um brief completo e acionável em menos de 3 minutos. Pronto para passar para produção ou para usar diretamente para criar o conteúdo na mesma sessão.
