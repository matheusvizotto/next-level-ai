# Inbox — Regras desta pasta

Captura rápida. Tudo que chega aqui precisa ser processado e movido.

## Regras
- Quando o usuário disser "anota", "captura" ou "salva por enquanto": jogue aqui
- Esta pasta não é destino final — é triagem
- Durante a revisão diária ou semanal, mova cada item para a pasta certa:
  - É sobre um projeto? → `03 Projects/`
  - É contexto permanente? → `02 Context/`
  - É reutilizável? → `04 Resources/`
  - É descartável? → delete
