# SEO — Fundamentos

> Template. Cobre o essencial de SEO clássico + AI search (GEO/AEO).

---

## SEO em 2026 mudou

**Antes (até 2023):**
- Google ranqueia páginas
- Você otimiza pra keyword
- Backlinks pesam muito

**Agora (2026):**
- Google + IAs (ChatGPT, Perplexity, Claude) ranqueiam e CITAM
- Você otimiza pra ser entendido por modelo de linguagem, não só por crawler
- E-E-A-T (Experience, Expertise, Authoritativeness, Trustworthiness) > backlinks isolados

**Implicação:** SEO clássico ainda funciona, mas precisa ser complementado com **GEO/AEO** (Generative/Answer Engine Optimization).

---

## A pirâmide de SEO

```
            E-E-A-T (autoria, credibilidade)
                  ▲
        Conteúdo profundo e útil
                  ▲
       Estrutura técnica (rápido, mobile, schema)
                  ▲
    Indexação e crawl (robots, sitemap, canonicals)
```

Subir na pirâmide só faz sentido se a base abaixo está sólida. Conteúdo brilhante em site lento e sem indexação = 0 tráfego.

## On-page essencial

| Elemento | Regra |
|---|---|
| Title tag | 50-60 caracteres, keyword no início, marca no fim |
| Meta description | 140-160 caracteres, com CTA implícito |
| H1 | Único, contém keyword principal |
| H2/H3 | Estrutura lógica, palavras relacionadas |
| URL | Curta, com keyword, sem parâmetros |
| Imagens | Alt text descritivo, WebP/AVIF, lazy-loading |
| Schema | Article / Product / FAQ / LocalBusiness conforme o caso |

Use `/seo-pagina` em qualquer URL pra auditoria automatizada.

## Conteúdo que ranqueia

**Princípios:**

1. **Intent match.** Pessoa busca "X" → ela quer guia, comparativo, ou compra? Sua página entrega o que ela quer?
2. **Profundidade > comprimento.** 1500 palavras úteis > 4000 palavras de filler.
3. **Estrutura escaneável.** H2/H3, bullets, tabelas, callouts. Pessoa scanea antes de ler.
4. **Original > copiada.** Google detecta similaridade. Mesmo título de 500 outros sites = nunca rankeia.
5. **Updated.** Conteúdo de 2022 sobre IA é morto. Atualize quando o tópico exige.

## GEO/AEO — otimização pra IAs

Quando ChatGPT/Perplexity citam fontes, eles preferem:

| Característica | Por quê |
|---|---|
| Schema markup completo | Modelos parseiam estrutura facilmente |
| Citações verificáveis | LLMs preferem fontes com data e link |
| Definições explícitas | "X é Y" frases curtas ranqueiam em respostas geradas |
| FAQ schema | Pergunta + resposta direta é formato favorito de IA |
| Autor com bio + credenciais | E-E-A-T atravessa pra IAs também |
| `llms.txt` na raiz do site | (Padrão emergente) IA sabe o que crawler pode usar |

**Estratégia prática:**
- Adicione FAQs no fim de cada artigo principal
- Use schema FAQPage
- Cite fontes com `<sup>` + link inline
- Crie bio com expertise específica em cada artigo

## Keywords

**Tipos:**
- **Short-tail:** "marketing" (volume alto, intent vago, competitivo)
- **Long-tail:** "como fazer marketing digital pra clínica de fisioterapia" (volume baixo, intent claro, conversor)

**Princípio:** 80% do esforço em long-tail. Short-tail vem como subproduto.

**Pesquisa de keyword:**
- Google Suggest ("...as pessoas também perguntam")
- AnswerThePublic
- SEMrush, Ahrefs (pagos)
- Google Search Console (mostra o que TE traz tráfego — gap = oportunidade)

---

## Links

**Internos:**
- Cada artigo importante linka pra 3-5 outros do mesmo cluster
- Anchor descritivo, não "clique aqui"

**Externos:**
- Saída pra fontes autoritativas (você não perde rank, ganha credibilidade)
- Backlinks bons: imprensa de nicho, parcerias, citações orgânicas
- Backlinks ruins: directories, comprados, link farms (penalizam)

---

## Métricas que importam

**No Google Search Console:**
- Impressões (você apareceu)
- CTR (clicaram)
- Posição média
- Páginas indexadas vs submetidas

**No site:**
- Tempo na página (intent match)
- Pages per session (conteúdo gera curiosidade)
- Conversion rate por landing (SEO + UX juntos)
