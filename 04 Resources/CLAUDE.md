# Resources — Regras desta pasta

Biblioteca pessoal reutilizável. Prompts, frameworks, referências, templates.

## O que vai aqui
- Prompts reutilizáveis (além dos que já estão em `prompts/`)
- Frameworks e metodologias de referência
- Swipe files — exemplos de conteúdo que funcionou
- Templates de documentos
- Arquivos de projetos concluídos

## Organização sugerida
```
04 Resources/
├── prompts/       ← prompts adicionais além do kit inicial
├── frameworks/    ← metodologias de referência
├── swipe/         ← exemplos e inspirações
├── templates/     ← modelos de documentos
└── arquivo/       ← projetos e conteúdos concluídos
```

## Regras
- Quando o usuário compartilhar um prompt, framework ou template reutilizável: salve aqui com nome descritivo
- Quando o usuário disser "salva isso para usar depois": roteie para cá
- Não crie subpastas vazias — crie a pasta só quando houver conteúdo para ela
