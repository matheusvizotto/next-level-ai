# Daily — Regras desta pasta

Notas diárias. Uma por dia, formato `YYYY-MM-DD.md`.

## Quando criar
- No início de cada sessão de trabalho, se não existir nota do dia

## Estrutura padrão
```markdown
---
type: daily-note
date: YYYY-MM-DD
status: active
tags: [daily]
---

## Foco do dia
- {{o que precisa ser feito hoje}}

## Sessão
- **Trabalhei em:** {{projeto ou tarefa}}
- **Resultado:** {{o que foi feito}}
- **Próximo passo:** {{o que vem depois}}

## Notas
{{qualquer captura rápida, ideia, decisão}}
```

## Regras
- Nunca edite uma nota de dia anterior — crie uma nova
- Ao capturar algo importante (decisão, insight, projeto novo), roteie para a pasta certa e deixe só um wikilink aqui
- Se o usuário disser "anota isso" ou "salva aqui", salve na nota do dia atual
